### Описание проекта    
Улучшить алгоритм, чтобы компьютер угадывал за еще меньшее количество попыток.

:arrow_up:[к оглавлению](_)


### Какой кейс решаем?    
Нужно улучшить программу, которая угадывает число за минимальное число попыток

**Условия соревнования:**  
- Компьютер загадывает целое число от 0 до 100, и нам его нужно угадать. Под «угадать», подразумевается «написать программу, которая угадывает число».
- Алгоритм учитывает информацию о том, больше ли случайное число или меньше нужного нам.
- Компьютер должен угадать менее чем за 20 попыток.

**Метрика качества**     
Результаты оцениваются по среднему количеству попыток при 1000 повторений

**Что практикуем**     
Учимся писать хороший код на python


### Краткая информация о данных
Первая версия программы предлагает человеку угадывать случайное число (numpy.random.randint(1, 101)).
Вторая версия программы угадывает сама, просто вызывая ту же функцию случайного числа. В среднем 101 попытка.
  
:arrow_up:[к оглавлению](.README.md#Оглавление)
